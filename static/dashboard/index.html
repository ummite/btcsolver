Ideas that maximize chances

  • Brainwallet: SHA256 + reverse + lower/upper + year suffixes
  • BIP39: multi-path + several indexes + forgotten passphrase
  • 11 known words + bruteforce the 12th (find-12th-word)
  • Personal phrases: emails, pets, dates, cities, « bitcoin 2011 »
  • WIF / hex recovered from old backups / screenshots
  • Electrum old / minikeys (repo CLI tools)
  • Always verify on-chain when Core is up to date
